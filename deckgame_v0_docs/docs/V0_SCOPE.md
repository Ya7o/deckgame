# V0_SCOPE.md

## Objectif

Prototype technique privé permettant de jouer une partie complète Core Set en local.

## Inclus

- 2 joueurs local pass-and-play.
- Web app Vite + React + TypeScript.
- Core Set uniquement.
- Trade Row, Explorer pile, Trade Deck.
- Deck de départ, pioche, défausse, reshuffle.
- Trade, Combat, Authority.
- Ships, Bases, Outposts.
- Ally abilities.
- Scrap.
- Effets complexes du Core Set.
- UI responsive mobile-first, placeholder.
- Tests Vitest.
- Log de partie.

## Exclu

- Online multiplayer.
- Backend.
- Comptes.
- Mobile natif.
- Thème Deckgame final.
- Nouvelles cartes.
- Extensions.
- Monétisation.
- Animations avancées.
- Drag and drop obligatoire.
- Bot intelligent.

## Critères de fin

- Une partie complète peut être jouée.
- Le moteur est séparé de l’UI.
- Les tests principaux passent.
- La victoire fonctionne.
- Les cartes sont données structurées.
- L’UI est utilisable en navigateur mobile.
- Le log permet de comprendre les actions.
