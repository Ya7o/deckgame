## Fonctionnalités V0.2

### Paysage plein écran

- Layout paysage dédié avec `data-layout="landscape"`
- Toutes les zones visibles sans scroll : adversaire, rangée commerciale,
  bases, MAIN, actions
- Bouton Activer direct sous chaque base non-épuisée (plus besoin de la modale)
- Fullscreen PWA supporté

### Résumé des actions du bot

- Barre `✦` visible pendant le tour humain suivant
- Affiche les N premières actions (3 en paysage, 4 en portrait)
- Si dépassement : compteur "+N actions" + bouton "Voir journal"
- Aucune information sur la main du bot divulguée

### Pédagogie

- Glossaire intégré (11 termes : Main, Pioche, Défausse, Écarter, Avant-poste…)
- Hints dans la modale CardDetail pour les bases, avant-postes, effets alliés
- Messages de blocage contextuels ("🛡 Avant-poste bloque l'attaque")
- Overlay UTILISÉ sur les bases épuisées

### Interface

- Hiérarchie boutons : Fin du tour = primaire dominant, Abandonner = secondaire discret
- États désactivés explicites (title tooltip pendant le tour bot)
- Portrait supporté et non régressé (tous tests verts à chaque patch)
